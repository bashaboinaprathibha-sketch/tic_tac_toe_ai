# tic_tac_toe_ai

**Description**

This project is a Python-based Tic Tac Toe game that allows a user to play against an AI opponent. The AI makes intelligent decisions using the Minimax algorithm, which evaluates all possible moves and chooses the best one to maximize its chances of winning while minimizing the opponent's chances.

**How It Works**
The game displays a 3×3 Tic Tac Toe board.
The human player enters a position to place their mark (X).
The AI player uses the Minimax algorithm to analyze all possible future game states.
The AI selects the optimal move (O) based on the evaluation.
The game continues until:
A player wins by placing three marks in a row, column, or diagonal.
The board is full, resulting in a draw.
The winner or draw result is displayed at the end of the game.
